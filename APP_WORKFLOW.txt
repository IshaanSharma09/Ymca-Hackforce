  STEP 1 — SIGN UP / LOGIN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  User opens the FitFuel app (PWA or browser).

  ┌─ NEW USER ─┐
  │ 1. Sign up with Google or Email via Firebase Auth               │
  │ 2. Onboarding wizard collects:                                  │
  │    • Name, Age, Gender                                          │
  │    • Height (cm), Weight (kg)                                   │
  │    • Activity level (Sedentary / Light / Moderate / Active)     │
  │    • Goal: Lose Fat / Build Muscle / Maintain / Rehab           │
  │    • Do you go to a gym? YES / NO                               │
  │    •   → If YES: select equipment your gym has (from checklist) │
  │    •   → If NO: bodyweight-only plans will be created           │
  │    • Do you have a health watch? Select: Samsung / Apple /      │
  │      Google Fit / None                                          │
  │ 3. App calculates BMR, TDEE, daily calorie target, macro split  │
  │ 4. Lands on the DASHBOARD                                       │
  └──────────────────────────────────────────────────────────────────┘

  ┌─ RETURNING USER ─┐
  │ 1. Login with Google/Email → straight to DASHBOARD              │
  └──────────────────────────────────────────────────────────────────┘

  STEP 3 — LOGGING MEALS (Breakfast / Lunch / Snack / Dinner)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  User taps [+ LOG MEAL] → chooses meal type (Breakfast / Lunch / Dinner / Snack)

  ┌─ OPTION A: SEARCH & ADD MANUALLY (primary method) ──────────────┐
  │                                                                  │
  │  User types: "Butter Chicken"                                    │
  │                                                                  │
  │  → App calls Foodoscope RecipeDB API:                           │
  │    GET /recipe2-api/recipe/recipesinfo?search=butter+chicken    │
  │    Authorization: Bearer <API_KEY>                              │
  │                                                                  │
  │  → Returns matching recipes with:                                │
  │    • Recipe_title: "Butter Chicken"                             │
  │    • Calories: 490 kcal (per serving)                           │
  │    • Protein: 38g                                                │
  │    • Carbs: 12g                                                  │
  │    • Fat: 32g                                                    │
  │    • Servings: 4                                                 │
  │                                                                  │
  │  → User selects recipe, chooses portions (1 serving)            │
  │  → App logs: 490 kcal, 38g protein, 12g carbs, 32g fat          │
  │                                                                  │
  │  FOR FULL NUTRITION DETAIL, app also calls:                     │
  │    GET /recipe2-api/recipe-nutri/nutritioninfo?recipeId=<id>    │
  │    → Returns per-ingredient: calcium, iron, vitamins, sodium,   │
  │      cholesterol, fiber, sugar, all fatty acids, etc.           │
  │                                                                  │
  │  AND for micronutrients:                                         │
  │    GET /recipe2-api/recipe-micronutri/micronutritioninfo        │
  │    → Returns: amino acids, all vitamins (A, B6, B12, C, D, E,  │
  │      K), minerals, phytosterols, etc.                           │
  │                                                                  │
  │  This is also how you ADD ITEMS HIDDEN FROM CAMERA — just       │
  │  type the dish name or ingredient manually and search.           │
  └──────────────────────────────────────────────────────────────────┘

  ┌─ OPTION B: BROWSE BY CATEGORY ──────────────────────────────────┐
  │                                                                  │
  │  User browses: Indian → North Indian → Vegetarian               │
  │                                                                  │
  │  → App calls Foodoscope:                                        │
  │    GET /recipe2-api/recipe/recipebycategory?Region=Indian       │
  │  → Shows filtered recipe list with calories                     │
  │  → User picks and logs                                           │
  └──────────────────────────────────────────────────────────────────┘

  ┌─ OPTION C: CALORIE / PROTEIN FILTER ────────────────────────────┐
  │                                                                  │
  │  User says: "Show me meals under 300 kcal with 20g+ protein"   │
  │                                                                  │
  │  → App calls:                                                    │
  │    GET /recipe2-api/recipe/recipebycalories?minCalories=0       │
  │        &maxCalories=300                                          │
  │    GET /recipe2-api/recipe/recipebyproteinrange?minProtein=20   │
  │                                                                  │
  │  → Shows matching healthy options → user picks and logs          │
  └──────────────────────────────────────────────────────────────────┘

  STEP 4 — STEP & ACTIVITY TRACKING (All Day, Automatic)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Running in the background all day:

  ┌─ WITH HEALTH WATCH (Samsung / Apple / Google Fit) ──────────────┐
  │  • Auto-syncs steps, heart rate, calories burned, SpO2, sleep   │
  │  • Data flows into FitFuel dashboard automatically               │
  │  • Example: 6,500 steps → 280 kcal burned (from watch data)    │
  └──────────────────────────────────────────────────────────────────┘

  ┌─ WITHOUT HEALTH WATCH (built-in tracking) ──────────────────────┐
  │  • Phone's accelerometer counts steps                            │
  │  • FitFuel estimates calories burned using MET formula:         │
  │    Calories = MET × weight(kg) × time(hours)                    │
  │  • Example: 6,500 steps × 0.04 kcal/step ≈ 260 kcal burned    │
  └──────────────────────────────────────────────────────────────────┘

  Dashboard updates:
    Steps: 6,500 / 10,000  ████████░░  65%
    Calories Burned (walking): 260 kcal

  HOW FOODOSCOPE API IS USED (Technical Flow)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Base URL: http://cosylab.iiitd.edu.in:6969
  Auth:     Authorization: Bearer <API_KEY>
  Rate:     25 requests per window

  RecipeDB Endpoints Used:
  ────────────────────────
  1. GET /recipe2-api/recipe/recipesinfo        → browse/search all recipes
  2. GET /recipe2-api/recipe/recipeofday        → recipe of the day feature
  3. GET /recipe2-api/recipe/recipe-day/with-ingredients-categories
                                                → recipe of day with filters
  4. GET /recipe2-api/recipe-nutri/nutritioninfo
                                                → macro nutrition per ingredient
  5. GET /recipe2-api/recipe-micronutri/micronutritioninfo
                                                → micro nutrition (vitamins,
                                                  minerals, amino acids)
  6. GET /recipe2-api/recipe/recipebycategory   → filter by region/cuisine
  7. GET /recipe2-api/recipe/recipebyid         → get specific recipe
  8. GET /recipe2-api/recipe/recipebyproteinrange → filter by protein
  9. GET /recipe2-api/recipe/recipebycalories   → filter by calorie range
  10. POST /recipe2-api/recipe/recipemealplan   → generate meal plans

  FlavorDB Endpoints Used:
  ────────────────────────
  1. Entity Controller     → get food entities and categories
  2. Food Pairing          → suggest food pairings for variety
  3. Molecule Controller   → flavor compound data
  4. Property Controller   → taste properties of ingredients


  Data Fields Available Per Recipe (from API):
  ─────────────────────────────────────────────
  • Recipe_title, Calories, Protein (g), Carbs (g), Fat (g)
  • cook_time, prep_time, total_time, servings
  • Region, Sub_region, Continent
  • Ingredients list with individual nutrition
  • 40+ micronutrients: vitamins A/B/C/D/E/K, iron, calcium,
    zinc, magnesium, sodium, cholesterol, fiber, sugar, all
    fatty acid types, amino acids, phytosterols, etc.
  • vegan / vegetarian / pescetarian flags
  • Utensils and Processes
